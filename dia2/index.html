Bienvenidos a mi pagina web

Un header o encabezado es una estructura que nos permite agrupar contenido introductorio o de navegacion.

La etiqueta main permite declarar cual es el contenido principal de la pagina web.


Por otro lado, el section servira como un contenedor que permitira estructura de mejor manera el contenido de la pagina web.
