Tipos de listas en HTML

Lista desordenada

Una lista desordenada es una lista en la que los elementos no tienen un orden específico. Se representa con la etiqueta ul para el contenedor y li para cada elemento de la lista.

  • Elemento 1
  • Elemento 2
  • Elemento 3

Lista ordenada

Una lista ordenada es una lista en la que los elementos tienen un orden específico. Se representa con la etiqueta ol para el contenedor y li para cada elemento de la lista.

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento

Lista de definición

Una lista de definición es una lista que se utiliza para definir términos y sus descripciones. Se representa con la etiqueta dl para el contenedor, dt para el término y dd para la descripción.

Término 1
Descripción del término 1
Término 2
Descripción del término 2
Término 3
Descripción del término 3
